Q5: What is the history of Unix or Linux?

Linux is a computer operating system created in the early 1990s by Linus Torvalds (a Finnish software engineer) and the FSF. In 1994 the Linux kernel that acted as the core of the operating system and around this time the FSF and an American software developer were trying to create an open source operating system similar to Unix (made in the 1960s). Unlike Linus who was working on the core components the FSF were working on utilities first and when these utilities were added to the Linux kernel that made a complete operating system.
